The Autonomous Coding Agent uses an AST (Abstract Syntax Tree) parsing loop and compiler warnings to guide the LLM's next action. When a compilation error is encountered:

1. Parse compiler logs for file line numbers.
2. Locate the failing code snippet using AST markers.
3. Feed the localized code + build stack trace back to the LLM.
4. Execute the revised code and run test suites.
